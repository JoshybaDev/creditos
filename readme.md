#### Sistema para la administración de creditos individuales
### Backend
- Desarrollado en Java con SpringBoot

### Frontend
- Desarrollado en Angular

### Funcionalidades
- Login con correo y contraseña
- Alta de clientes
- Edición de clientes
- Listar clientes
- Alta de créditos
- Autorizar y Cancelar Créditos
- Ministrar Créditos
- Listar Cuotas por credito_id
